#  MERN Task Manager

A beginner-friendly **MERN Stack Task Manager** application built using **React (Vite)**, **Node.js**, **Express**, and **MongoDB**.  


## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using MongoDB
- RESTful API architecture



## Tech Stack

### Frontend
- React (Vite)
- JavaScript
- Fetch API

### Backend
- Node.js
- Express.js
- Mongoose

### Database
- MongoDB (Local or Atlas)
